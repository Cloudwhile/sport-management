# 环境变量配置说明

## 配置文件

前端项目使用 `.env` 文件来管理环境变量配置。

### 文件说明

- `.env` - 实际的环境变量配置文件（不会提交到 Git）
- `.env.example` - 环境变量配置示例文件（会提交到 Git）

## 可用的环境变量

### VITE_APP_TITLE

应用程序的标题，会显示在：
- 登录页面标题
- 导航栏标题
- 浏览器标签页标题
- Dashboard 欢迎语
- 登录页版权信息

**默认值**: `学校体测数据管理系统`

**示例**:
```env
VITE_APP_TITLE=广东第二示范学院附属中学体测数据管理系统
```

### VITE_API_BASE_URL

API 请求的基础路径。

**默认值**: `/api`

**示例**:
```env
VITE_API_BASE_URL=/api
```

## 使用说明

1. 复制 `.env.example` 文件为 `.env`
2. 修改 `.env` 文件中的配置项
3. 重启开发服务器以使配置生效

## 注意事项

- `.env` 文件已被添加到 `.gitignore`，不会被提交到版本控制
- 环境变量必须以 `VITE_` 开头才能在前端代码中访问
- 修改环境变量后需要重启开发服务器
